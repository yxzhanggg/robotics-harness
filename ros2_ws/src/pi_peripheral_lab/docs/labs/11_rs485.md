# 11 RS485

## Objective / Goal

Transmit a simple CRC-protected teaching frame over half-duplex RS485 from
atlas to vector.

## Learning Points

Differential pair A/B, half-duplex direction control, termination resistor,
bias resistor, bus topology, and CRC checking.

## BOM

atlas and vector, two RS485 transceiver modules, twisted pair, 120 ohm
termination resistor, bias resistors if modules do not include them.

## Circuit Diagram

```text
atlas UART TX/RX/DE ---- RS485 transceiver ---- A/B twisted pair ---- transceiver ---- vector UART RX/TX/DE
atlas GND ------------------------------------- reference ---------------------------- vector GND
```

## Wiring Table

| Signal | Connection |
| --- | --- |
| A | atlas transceiver A to vector transceiver A |
| B | atlas transceiver B to vector transceiver B |
| GND | atlas GND to vector GND reference |
| UART side | Pi 3.3 V UART to transceiver logic side |

## Oscilloscope Setup

Probe A and B with a differential probe or two channels and math subtraction.
Confirm differential transitions during frames.

## Function Generator Setup

Not used.

## Run Commands

On vector:

```bash
ros2 launch pi_peripheral_lab vector_rs485_receiver.launch.py dry_run:=false
ros2 topic echo /pin_lab/rs485/rx
```

On atlas:

```bash
ros2 launch pi_peripheral_lab atlas_rs485_sender.launch.py dry_run:=false
ros2 topic echo /pin_lab/rs485/tx
```

## Expected Waveform

A/B differential pair toggles during frame transmission. Receiver reports
`crc_ok: true`.

## Common Mistakes

- A/B swapped.
- No termination on long cables.
- Direction control not wired or configured.
- 5 V transceiver logic connected directly to GPIO.

## Safety Checklist

- Transceiver logic side is 3.3 V compatible or level shifted.
- Bus termination and bias are understood.
- Common reference is safe.

## HIL Pass/Fail Criteria

Pass: vector reports payload with valid CRC and scope shows differential signal.

Fail: CRC failures persist, no differential signal, or transceiver logic levels
are unsafe.

## Engineering Reuse

Teaching experiment: RS485 sender/receiver nodes.

Reusable module: `protocol_bits.rs485_lab_frame` and CRC16 helper.
